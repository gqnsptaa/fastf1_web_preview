FastF1 Web Preview

Drop your generated plot as plot.png and view it here.

How to use

  1. Run your Python script to generate a plot image: 
    # example snippet in your Python script
import fastf1, fastf1.plotting as f1plt
import matplotlib.pyplot as plt

fastf1.Cache.enable_cache(".fastf1")

session = fastf1.get_session(2021, 7, "Q")
session.load(laps=True)

f1plt.setup_mpl()  # optional: nice style
fig, ax = plt.subplots()
session.laps.pick_fastest().telemetry.plot(x="Distance", y="Speed", ax=ax)
ax.set_title(f"{session.event['EventName']} — {session.name}")
fig.savefig("plot.png", dpi=150, bbox_inches="tight")
print("Saved plot.png")
  2. Put this index.html and styles.css in the same folder as your script.
  3. Open index.html in your browser (or use VS Code Live Server).

Plot preview

If the image doesn't show, make sure your Python script saved plot.png in this folder.

Tips

  • Keep plots next to this file, or change the src to your actual image path.
  • For multiple images, duplicate the <img> element or create more cards.
  • Want interactivity? Save Plotly figures with fig.write_html('plot.html') and link them here.
